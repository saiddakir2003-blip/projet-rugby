APPLICATION RUGBY U15 - VERSION AMELIOREE ENTRAINEMENT

Contenu :
- Rugby_U15_application_finale.html
- videos/ : videos de demonstration collective
- downloads/ : fiche d'observation, modele CSV et exemples

Nouveautes de cette version :
- Entrainement collectif plus detaille.
- Chaque axe contient maintenant : organisation, consignes, criteres de reussite, variables N1/N2/N3, securite/adaptation U15, PPI, dosage et video.
- Le modele CSV telechargeable a ete aligne avec le modele attendu par l'application.

Utilisation :
1. Decompresser le ZIP.
2. Ouvrir Rugby_U15_application_finale.html dans Chrome ou Firefox.
3. Garder les dossiers videos et downloads dans le meme dossier que le fichier HTML.
4. Importer un CSV dans l'onglet Aspect individuel.
5. Ajouter plusieurs joueurs a l'equipe.
6. Cliquer sur Analyser l'equipe pour afficher l'entrainement collectif detaille.
