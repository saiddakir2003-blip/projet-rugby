# Rugby U15 — Site web prêt à partager

Cette version contient l'application Rugby U15 dans un seul fichier `index.html`.

## Utilisation locale
Ouvrir directement `index.html` dans Chrome, Edge ou Firefox.

## Déploiement GitHub Pages
1. Ouvrir le dépôt GitHub.
2. Cliquer sur **Add file** puis **Upload files**.
3. Déposer le fichier `index.html` à la racine du dépôt.
4. Cliquer sur **Commit changes**.
5. Aller dans **Settings > Pages**.
6. Choisir :
   - Source : Deploy from a branch
   - Branch : main
   - Folder : /root
7. Cliquer sur **Save**.
8. Attendre quelques minutes.
